About DRIP

DRIP — Daily Rootzone Irrigation Planner is a field-by-field irrigation scheduling tool that combines daily weather data with a soil water balance model to help farmers decide when and how much to irrigate.

Weather data is retrieved from Open-Meteo (open-source, CC BY 4.0). Soil texture auto-detection uses the USDA Web Soil Survey dataset. Evapotranspiration is estimated using the FAO-56 Penman-Monteith method.

Developed at Kansas State University. For questions or feedback, contact the authors.
