# DRIP: Daily Rootzone Irrigation Planner

A browser-native scheduling tool that combines real-time weather data, a soil water balance model, and AI-powered recommendations to help irrigators make informed and timely decisions.

## Usage

Open `index.html` in any browser — no installation required.

1. Set your field location, crop, and soil
2. Adjust irrigation system settings as needed
3. Click **Generate Schedule**
4. Review the soil water balance chart and daily schedule table
5. Read the AI-generated season outlook

## Crops

Corn · Soybean · Grain Sorghum · Winter Wheat · Sunflower
